# Product Overview

React + TypeScript + Nitro Full-Stack Starter - A production-ready boilerplate for building modern full-stack applications with React on the frontend and Nitro for the backend API.

## Purpose

Provides a complete full-stack development environment that eliminates initial setup time. Includes both client-side and server-side file-based routing, UI components, code quality tools, and Docker support out of the box.

## Key Features

### Frontend

- File-based routing with automatic route generation (vite-plugin-pages)
- shadcn/ui component library integration
- Auto-import for React hooks and components
- SVG-as-component support
- Google Fonts integration

### Backend

- Nitro 3 server with H3 handlers
- File-based API routing in `/routes` directory
- TypeScript support for API handlers
- Fast development with HMR

### DevOps

- Docker deployment ready
- Git hooks for code quality enforcement
- ESLint + Prettier configured
- Husky pre-commit hooks
